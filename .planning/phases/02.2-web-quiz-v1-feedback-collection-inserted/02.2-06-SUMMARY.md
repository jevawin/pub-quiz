## Accomplishments
- Play screen renders questions with 4 clickable options, shows correct/incorrect + explanation on reveal
- Four feedback buttons (Good/Bad/Confusing/Next) each record a question_plays row via Supabase
- recordQuestionPlay helper uses insert-only pattern (no .select()) with outbox fallback
- Deep-link protection redirects /play without router state to /
- Timer pauses on answer, resets per question
- Navigates to /done after last question with score and answers
