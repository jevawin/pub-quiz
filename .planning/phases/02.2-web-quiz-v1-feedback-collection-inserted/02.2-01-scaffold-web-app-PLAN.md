1. `mkdir -p apps/web && cd apps/web` and initialise manually (do not run `npm create vite` interactively — write files directly). Create `package.json` with:
   - `"name": "pub-quiz-web"`, `"private": true`, `"type": "module"`, `"version": "0.0.0"`
   - scripts: `"dev": "vite"`, `"build": "tsc -b && vite build"`, `"preview": "vite preview"`, `"test": "vitest"`, `"test:run": "vitest run"`
   - dependencies: `react@^19`, `react-dom@^19`, `react-router-dom@^7`, `@supabase/supabase-js@^2.101.1`, `class-variance-authority`, `clsx`, `tailwind-merge`, `@radix-ui/react-radio-group`, `@radix-ui/react-label`, `@radix-ui/react-slot`, `lucide-react`
   - devDependencies: `vite@^6`, `@vitejs/plugin-react@^4`, `typescript@^5.5`, `@types/react`, `@types/react-dom`, `vitest@^2`, `@testing-library/react@^16`, `@testing-library/jest-dom@^6`, `jsdom@^25`, `tailwindcss@^3.4`, `postcss@^8`, `autoprefixer@^10`

2. `tsconfig.json` — strict mode on, `"jsx": "react-jsx"`, `"moduleResolution": "bundler"`, paths alias `"@/*": ["./src/*"]`. `tsconfig.node.json` for vite config.

3. `vite.config.ts`:
```ts
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import path from 'path';
export default defineConfig({
  plugins: [react()],
  resolve: { alias: { '@': path.resolve(__dirname, './src') } },
});
```

4. `vitest.config.ts`:
```ts
import { defineConfig } from 'vitest/config';
import react from '@vitejs/plugin-react';
import path from 'path';
export default defineConfig({
  plugins: [react()],
  resolve: { alias: { '@': path.resolve(__dirname, './src') } },
  test: {
    environment: 'jsdom',
    setupFiles: ['./src/test/setup.ts'],
    globals: true,
  },
});
```

5. `src/test/setup.ts`:
```ts
import '@testing-library/jest-dom/vitest';
import { afterEach, beforeEach, vi } from 'vitest';
beforeEach(() => {
  localStorage.clear();
  vi.useRealTimers();
});
afterEach(() => { vi.restoreAllMocks(); });
```


  - `label.tsx` (Label wrapping @radix-ui/react-label)
