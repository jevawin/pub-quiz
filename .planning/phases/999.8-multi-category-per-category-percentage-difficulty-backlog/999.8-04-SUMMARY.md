## Decisions Made

- `calibrateQuestion` requires 5 parameters (including claude client and config). The backfill script instantiates these internally rather than requiring callers to pass them — cleaner CLI interface.

- Updated the Wave 0 test to match the UPDATE-based implementation per plan instructions.
